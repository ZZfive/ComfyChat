## Overview

An experiment about combining multiple LoRAs with DARE [https://arxiv.org/pdf/2311.03099.pdf](https://arxiv.org/pdf/2311.03099.pdf)

## Features

* Merge LoRA Stacks
* Save LoRA
* Experimental lipschitz regularizer (set -1 to turn off)
